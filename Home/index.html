A InvestUSP

O InvestUSP procura trabalhar como uma ponte entre os pesquisadores de dentro da faculdade e as empresas que procuram fomentar esses projetos.

Para impedir que apenas algumas áreas recebam investimento, parte do financiamento desses projetos é direcionado para os funcionários da USP e para um fundo de Endowment para ajudar também a pesquisa de base.

O fundo de Endowment foi criado para se impedir que, caso acabe o investimento das empresas, ainda seja possível que algum fluxo de dinheiro seja direcionado para os funcionários da USP
